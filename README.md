# Steam-collection-size-calculator
Calculates the size of your steam mod collection. Ive been in that situation, wanting to download a game with mods, but was unsure whether I had enough space, well, now I have a good idea how much storage imma need before downloading something.

Requirements/ Prerequisites:
-  have Python/ Pycharm
-  Ensure your mod collection is set to "public" (Unlisted might work im not sure)

  Usage:

COLLECTION_ID = "INPUT MOD COLLECTION ID HERE"
this is on line 6, basically, get the ID for your mod collection, for example:
https://steamcommunity.com/sharedfiles/filedetails/?id=3716855116


"3716855116" is your mod collection ID. 
so itd look something like this:

COLLECTION_ID = "3716855116"

thats it, literally grab the steam collection ID, and paste the numbers into it and run the script. It does take a bit of a while to fetch the sizes, so be patient.
